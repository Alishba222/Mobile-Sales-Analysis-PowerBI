# Mobile Sales Performance Dashboard
## Project Overview:
This project provides a comprehensive analysis of mobile sales data across different brands, cities, and payment methods. The dashboard helps stakeholders understand sales trends, customer satisfaction, and inventory movement.

## Key Features:
Sales Overview: Total sales reached 308M with over 2K transactions.

Brand Performance: Comparison between top brands like Apple, Samsung, OnePlus, Vivo, and Xiaomi.

Geographical Analysis: Sales distribution across major Indian cities (Delhi, Mumbai, Hyderabad, etc.).

Customer Insights: Funnel chart showing customer ratings and trend lines for monthly sales.

Payment Trends: Breakdown of transactions via UPI, Credit Card, Debit Card, and Cash.

## Tech Stack Used:
Tool: Power BI Desktop

Data Source: Excel / CSV

Visuals: Map visuals, Funnel charts, Pie charts, and Line graphs.

## How to View:
Download the .pbix file from this repository.

Open it using Power BI Desktop.
